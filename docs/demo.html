Accordion (nativo con details/summary)

¿Qué es RenDS?

Un design system vanilla, accesible y atómico que sirve como punto de partida para cualquier proyecto. Sin frameworks, sin dependencias.

¿Necesito React?

No. RenDS es 100% vanilla HTML, CSS y JavaScript. Los Web Components funcionan en cualquier framework o sin ninguno.

¿Cómo maneja dark mode?

Automáticamente con prefers-color-scheme, o manualmente con data-theme="dark" en el <html>.
